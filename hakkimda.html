Hello World!

Merhaba, ben Ahmet Ekmekcioğlu. 11 Şubat 2009 tarihinde İstanbul'da dünyaya geldim. Çocukluk yaşlarımın büyük çoğunluğunu bir şeyleri kurcalayarak geçirdim. Bu kurcalama hevesim Ortaokul yıllarımda Wordpress üzerinden web sitesi kurmamla buralara kadar geldi. Önce "Web sitesi nasıl kuruluyor acaba?" sorusunu sormuştum. Ardından "Web siteleri nasıl kodlanıyor acaba?" diyerek yazılım merakımın fitilini ateşlemiştim. Çeşitli dillerde üstünkörü bilgiye sahibim fakat tekrar o yıllardaki gibi HTML ile başlamaya karar verdim ve şimdi buradayım.

Yalnız üstünkörü bilgi sahibi olduğum dillerde proje geliştirecek seviyede değilim. Şimdilik üstünkörü bilgi sahibi olduğum diller şunlar:

  • HTML
  • CSS
  • JavaScript
  • C#
  • Python

Daha önceden de söylediğim gibi bu dillerde proje geliştirebilecek seviyede değilim fakat halen öğreniyorum.
